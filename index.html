Cutting Edge Frontend

SVG

The web is currently mostly lacking vector graphics. Web pages are accessed with varying screen resolutions, but for example icons are rendered using fixed size PNG images, that are optimal for the largest resolution. Browser games that use vector graphics are implemented with proprietary Adobe Flash.

The W3C standard Scalable Vector Graphics is being more widely supported every day with all major browser vendors finally supporting it.

LaTTeX HTML Typesetting

Currently any serious publishing needs to be done with LaTeX or similar typesetting system, that for example takes care of references and provides automatic and consistent styling.

LatteX is a project to find the limits of HTML5+CSS3+JavaScript when implementing the full functionality of LatteX in browser. By targetting this goal, we understand better what it means to publish documents in web without PDFs and learn from browser support and their limitations.

http://github.com/matti/lattex
